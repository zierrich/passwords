## Password generation rules:

1. Ten passwords are generated, each 18 characters long.

2. The first character is always a letter.

3. The password contains exactly four special characters from the set !@#$%&*:;[]{}?.

4. Letters (a–z, A–Z) may appear consecutively for 1 to 3 characters.

5. Numbers (0–9) may appear consecutively for 1 to 2 characters.

6. Special characters are never consecutive.

7. The same character cannot appear consecutively.

8. The remaining characters are distributed randomly, but all of these restrictions are taken into account.
